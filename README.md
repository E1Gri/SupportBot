Бот с RAG системой 

Перед запуском необходимо добавить следущие файлы:

info.txt - Файл с размеченной информацией (Пример уже добавлен)
token.txt - Файл, где записан токен Бота в ТГ в 1 строке
admins.txt - Файл, где записаны телеграм id пользователей, которые выполняют роль операторов. Каждый id на отдельной строке
api.txt - Файл, где хранится API  GigaChat в 1 строке

Все необходимые пакеты для загрузки указаны в requirements.txt
